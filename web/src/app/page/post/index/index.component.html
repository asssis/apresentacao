Instalando o postgresql ubuntu

> sudo apt-get update

> sudo apt-get install postgresql postgresql-contrib

Configurando postgresql

> su - postgres

> psql

> createuser --interactive --pwprompt

Enter name of role to add: cleopatra Enter password for new role: Enter it again: Shall the new role be a superuser? (y/n) y

> createdb -O cleopatra egypt

> sudo nano /etc/postgresql/10/main/postgresql.conf substituir #listen_addresses = 'localhost' listen_addresses = '*'

> sudo nano /etc/postgresql/10/main/pg_hba.conf substituir host all all 127.0.0.1/32 md5 host all all 0.0.0.0/0 md5

ativar firewall > sudo ufw allow 5432/tcp

> sudo systemctl restart postgresql > psql -h {{'{server_ip}'}} -d egypt -U cleopatra
